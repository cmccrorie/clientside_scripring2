What is Petal

Petal is a WYSIWYG editor aiming to make use of JDITA and Prosemirror to create and edit LwDITA documents on the web.

Petal allows the user to: See a readable HTML representation of the document, Edit the text, Insert new elements, Transform content, Edit attributes.
